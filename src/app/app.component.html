• CRUD App project features/Operations:
    • List car number plates with their owner names.
    • Add new car number plates.
    • Remove existing car number plates.
    • Update existing car number plates.
    • View car number plate details.
  • Validations:
    • Car number plate validation (must be exactly 6 characters: first 3 letters (A-Z), last 3 numbers).
    • Only unique car number plates are allowed.
  • Future features to be added:
    • Owner name validation (must be at least 3 characters).
    • Clear all car number plates.
    • Save records to local storage.
    • Integration with a backend server.
